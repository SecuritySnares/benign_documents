eHub: Action Items

Overview

You can leverage the Action Items communications solution across your suite of TEAM Products to capture, escalate, track and audit previously untraceable email/phone/text messages or in-person conversations, both in and from the field. When an Employee, Supervisor or Customer submits an Action Item Request (or when TeamTime automatically submits an Alert), the Action Items communications solution captures the information and automatically initiates a notification and escalation process.

Employee Self Service (ESS)

Employees (Supervisor or Employee license levels) can create, view and edit Action Items via eHub for their Primary Job and Jobs for which they are scheduled in Personnel Scheduling. An Employee level access must be a participant in order to create, view and edit an item. Supervisor level access can create, view and edit Action Items for their primary job, jobs they supervise (hierarchy), and for full access Quality Assurance Jobs. Supervisor can edit ONLY those items in which they are a participant .

Note:  WinTeam users can also create/view/manage Action Items from the SYS:Action Item and SYS:Action Items Review screen.

Customer Self Service (CSS)

Customers can create Action Items for jobs that are in their standard job list setup in WinTeam AR: Customer Master File - Contacts tab. They can view/edit ONLY those items on which they are a participant.

All changes made by a user within a single session (currently only available in eHub Web and eHub Mobile), are stored as a single history record and a single notification is sent out for this.

Key Functionality

Access the Action Items Screen

For an ESS - Supervisor level access license the Action Items widget is accessed from the Main, Job or My eHub landing pages.

For an ESS - Employee level access license the Action Items widget is accessed from the Job or My eHub landing pages.

For a CSS - Customer Self Service access license the Action Items widget is accessed from the Main or Job landing pages.

This widget displays the count of Open Items and My Items. It also has a Create New button which displays a window to allow you to create a new Action Item.

The ESS - Employee level access and CSS users will not see the Open Items button.

View Items You are a Participant in

On the widget, click View All or My Item to navigate to the Action Items screen to display the Action Item cards that you are a participant in.

View Open Items

On the widget, click Open Item to navigate to the Action Items screen to display the list of items which are open. When a job's supervisor is changed (either manually or using the Supervisor Transition tool) the new supervisor has the ability to view and edit the action items in the Open tab. Any user with security access to view action items in the Open tab can also edit them. After a user make an edit to an action item, that user is automatically subscribed to future updates for that action item.

Note:  The ESS - Employee level access and CSS users do not see the Open tab.

View Closed Action Items

From the Action Items screen, select the Closed tab to display a list of closed Action Items.

Export Action Items

Click the Export icon in the top right corner of the window to export the information into a spreadsheet. The spreadsheet is automatically saved as a Microsoft Excel Worksheet (.xlsx) to the default download location on your computer or network.

Display Action Item History
  1. From the Action Items screen, select a card to expand and display the details.
  2. Click on the View History link to display the history of the Action Item.

Click on the View History link again to collapse the history details.

View an Image Attached to an Action Item
  1. From the Action Items screen, select a card to expand and display the details.
  2. Click the image icon to display the images in the File Viewer window.

Use the arrows to scroll to additional images if appropriate.

To close the File Viewer screen, click the Close button.

For information on how to upload an image to an Action Item, see eHub: Adding Images to an Action Item.
