action item

Action items are a list of "to do" tasks for managing projects. If you have the Project Management application, you can develop action items for managing details. For example, you can have action items for managing such items as:

  • leases
  • SDSs
  • commissioning -- action items structure the design submission approval process, and to generate action items that are assessment items for each piece of equipment you are evaluating as part of the commissioning process.
  • Clean Building tasks

One purpose of action items is to store cost and scheduling information for a project by the project phase; that is, cost and schedules are entered as:

  • baseline estimates when projects are requested
  • design estimates during the planning phase
  • actual costs and duration when the project is executed.

In this way, action items enable you to track and analyze time and work estimates and compare them to actual time and costs. Action items can also be combined into work packages that are put out for bidding to vendors.
